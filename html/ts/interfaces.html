    Interfaces


  • Que es una interfaz?

    • En TypeScript, una interfaz es una estructura que define un contrato para los objetos. Describe la forma que deben tener los objetos, especificando qué propiedades y métodos deben estar presentes y sus tipos. Las interfaces permiten garantizar que los objetos cumplan con una estructura específica, facilitando la validación de tipos y la documentación del código.

  • sintaxis de una instancia
    • interface NombreInterfaz { propiedad1: tipo1; propiedad2: tipo2; metodo1(parametro: tipo): tipoDeRetorno; }
  • ejemplo
    • interface Persona { nombre: string; edad: number; saludar(): void; }

    Algunos casos de uso

  • Definir estructuras de datos

    • Las interfaces son útiles para definir la estructura de datos en aplicaciones. Esto ayuda a asegurar que los objetos cumplen con una forma específica y facilita la validación de datos.

    interface Producto { id: number; nombre: string; precio: number; } const producto: Producto = { id: 1, nombre: "Laptop", precio: 1200 };
  • Modelar Configuraciones

    • Las interfaces permiten definir contratos que las clases deben cumplir, garantizando que las clases implementen las propiedades y métodos necesarios.

    interface Vehiculo { marca: string; modelo: string; acelerar(): void; } class Coche implements Vehiculo { constructor(public marca: string, public modelo: string) {} acelerar() { console.log("El coche está acelerando."); } }
