    Clases y Métodos


  • Que es una clase?

    • Una clase es una estructura predefinida que sirve como molde para crear Objetos. En este «molde» se declaran atributos y métodos; un atributo puede ser una variable, y un método puede ser una función, si lo comparamos con Javascript, a continuación se vera un ejemplo de la estructura de una clase.

    class Persona { nombre: string; edad: number; constructor(nombre: string, edad: number) { this.nombre = nombre; this.edad = edad; } saludar() { console.log("Hola, mi nombre es " + this.nombre); } }
  • Que es un método?

    • Los métodos son funciones asociadas a una clase, en la definición de clase se dijo que una clase como un molde para crear objetos Cada objeto creado a partir de esta clase (instancia) tendrá sus propias propiedades y podrá realizar ciertas acciones. Los métodos son esas acciones específicas que un objeto puede llevar a cabo como se puede ver en el siguiente ejemplo.

    class Coche { color: string; modelo: string; constructor(color: string, modelo: string) { this.color = color; this.modelo = modelo; } acelerar() { console.log("El coche está acelerando"); } frenar() { console.log("El coche está frenando"); } } // Creamos dos instancias de la clase Coche let coche1 = new Coche("rojo", "Ferrari"); let coche2 = new Coche("azul", "Ford"); // Llamamos a los métodos de cada instancia coche1.acelerar(); // Imprime: El coche está acelerando coche2.frenar(); // Imprime: El coche está frenando
    En este ejemplo:
  • Clase Coche: Define un molde para crear objetos tipo coche, con propiedades como color y modelo.
  • Métodos acelerar() y frenar(): Son acciones que un coche puede realizar. Están asociados a la clase Coche, por lo que cualquier objeto creado a partir de esta clase podrá utilizar estos métodos.
  • Instancias coche1 y coche2: Son objetos concretos creados a partir de la clase Coche. Cada uno tiene sus propias propiedades (color y modelo) y pueden utilizar los mismos métodos (acelerar y frenar).

    • Por que son importantes los métodos en las clases


  • Encapsulación: Los métodos permiten encapsular la lógica interna de un objeto, ocultando la implementación y exponiendo solo la funcionalidad necesaria.
  • Reutilización de código: Un mismo método puede ser utilizado por múltiples instancias de una clase, evitando la duplicación de código.
  • Programación orientada a objetos: Los métodos son fundamentales para el paradigma de la programación orientada a objetos, ya que permiten modelar el comportamiento de los objetos de manera natural.
