    Que Es Una Variable?

    Una variable representa un contenedor o un espacio en la memoria física o virtual de una computadora, donde se almacenan distintos tipos de datos (valores) durante la ejecución de un programa. A cada variable se le asigna un nombre descriptivo o un identificador que se refiere al valor guardado. Los datos almacenados pueden cambiar de valor o ser constantes.


    Tipado

  • Estatico

    • los tipos de tipado estatico son aquellos en donde se especifica que dato va a ingresar en la variable los tipados que soporta TypeScript son los siguientes

    boolean - valores verdaderos o falsos number - números enteros y valores en coma flotante string - valores de texto como "TypeScript Rocks"
  • Inferencia de tipos

    • TypeScript puede inferir el tipo de una variable en muchos casos, como se puede ver en el siguiente ejemplo

    let mensaje = "Hola"; // Ts infiere que mensaje es tipo string

    Sintaxis

    La sintaxis de TypeScript es muy similar a la de JavaScript, pero con algunas adiciones para soportar características de tipado y orientación a objetos, se puede ver con el siguiente ejemplo.

    let nombre: string = "Juan"; let edad: number = 30; let esEstudiante: boolean = true;
