    Conceptos Básicos

    a continuación se mostrara los conceptos básicos de typescript,conceptos tales como ciclos,condicionales,métodos y funciones etc, los cuales son esenciales para el trabajo de este lenguaje de programación


    Ciclos

  • For

    • El ciclo for se utiliza cuando se conoce de antemano el número de iteraciones que se deben realizar. Generalmente se usa para iterar sobre una secuencia de valores, como los elementos de una lista o los números en un rango.

    for (let i = 0; i < 10; i++) { console.log(i); }
  • while

    • El ciclo while se utiliza cuando el número de iteraciones no se conoce de antemano y el ciclo debe continuar mientras se cumpla una condición.

    let i = 0; while (i < 10) { console.log(i); i++; }
  • Do-While

    • El ciclo do-while es similar al while, pero la condición se evalúa después de que el código dentro del ciclo se ha ejecutado. Esto asegura que el código dentro del ciclo se ejecute al menos una vez.


    let i = 0; do { console.log(i); i++; } while (i < 10);

    condicionales

    Los condicionales permiten a los programas tomar decisiones basadas en ciertas condiciones. Aquí se explica los condicionales más comunes en TypeScript, junto con ejemplos:

  • if

    • El condicional if se utiliza para ejecutar un bloque de código si se cumple una condición.

    if (condición) { // Código a ejecutar si la condición es verdadera } //ejemplo const num = 10; if (num > 5) { console.log("El número es mayor que 5."); }
  • if-else

    • El condicional if-else permite ejecutar un bloque de código si la condición es verdadera y otro bloque si es falsa.

    const num = 3; if (num > 5) { console.log("El número es mayor que 5."); } else { console.log("El número no es mayor que 5."); }
  • switch

    • El condicional switch se utiliza para realizar múltiples comprobaciones en una sola variable. Es útil cuando se tienen muchas condiciones basadas en el mismo valor.

    const color = "rojo"; switch (color) { case "rojo": console.log("El color es rojo."); break; case "verde": console.log("El color es verde."); break; case "azul": console.log("El color es azul."); break; default: console.log("Color desconocido."); }

    Funciones y Metodos

  • función

    • Las funciones en TypeScript son bloques de código reutilizables que realizan una tarea específica. Se definen con la palabra clave function y pueden aceptar parámetros y devolver valores.

    function saludar(nombre: string): string { return `Hola, ${nombre}!`; } const mensaje = saludar("Juan"); console.log(mensaje); // Imprime: Hola, Juan!
  • Funciones Anónimas y Lambda

    • Las funciones anónimas son funciones sin nombre y se pueden definir usando la sintaxis de función flecha (=>).

    const suma = (a: number, b: number): number => { return a + b; } const resultado = suma(5, 3); console.log(resultado); // Imprime: 8
  • Métodos/span>

    • En TypeScript, un método es una función que está asociada a un objeto, generalmente dentro de una clase. Los métodos definen el comportamiento de los objetos creados a partir de una clase.

    class Persona { nombre: string; constructor(nombre: string) { this.nombre = nombre; } saludar(): string { return `Hola, soy ${this.nombre}`; } }
  • Uso de métodos
    • const persona = new Persona("Ana"); console.log(persona.saludar()); // Imprime: Hola, soy Ana

    Arrays

    los arrays (o arreglos) son colecciones de elementos del mismo tipo que se almacenan en una secuencia ordenada. Los arrays son muy útiles para manejar listas de datos. A continuación, se explica cómo trabajar con arrays en TypeScript con ejemplos:

  • sintaxis de un array
    • let numeros: number[] = [1, 2, 3, 4, 5]; let nombres: Array = ["Ana", "Luis", "Carlos"];
  • acceso a elementos

    • Puedes acceder a los elementos de un array utilizando índices. Los índices en los arrays son cero-basados.

    let colores: string[] = ["rojo", "verde", "azul"]; console.log(colores[0]); // Imprime: rojo console.log(colores[2]); // Imprime: azul
  • Modificación de Elementos

    • Puedes modificar los elementos de un array asignando un nuevo valor a un índice específico.

    let frutas: string[] = ["manzana", "banana", "naranja"]; frutas[1] = "kiwi"; console.log(frutas); // Imprime: ["manzana", "kiwi", "naranja"]
  • Métodos de Arrays

    • Los arrays en TypeScript tienen muchos métodos útiles. Aquí algunos ejemplos:

    push(): //Añade uno o más elementos al final del array. let numeros: number[] = [1, 2, 3]; numeros.push(4); // Añade 4 al final console.log(numeros); // Imprime: [1, 2, 3, 4] pop(): //Elimina y devuelve el último elemento del array. numeros.pop(); // Elimina el último elemento console.log(numeros); // Imprime: [1, 2, 3] shift(): //Elimina y devuelve el primer elemento del array. let letras: string[] = ["b", "c", "d"]; letras.unshift("a"); // Añade 'a' al principio console.log(letras); // Imprime: ["a", "b", "c", "d"] unshift(): //Añade uno o más elementos al principio del array. letras.shift(); // Elimina el primer elemento console.log(letras); // Imprime: ["b", "c", "d"] forEach()//: Ejecuta una función para cada elemento del array. let numeros: number[] = [1, 2, 3, 4, 5]; numeros.forEach(num => console.log(num * 2)); // Imprime: 2 4 6 8 10
