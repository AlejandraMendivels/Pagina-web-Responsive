    Instalacion Angular


    A continuacion se vera la manera de instalar angular paso a paso


    Instalar desde cli

    Antes de comenzar, asegúrate de tener instalados: Node.js y npm: Son fundamentales para gestionar paquetes en JavaScript y ejecutar aplicaciones Node.js. Puedes descargarlos desde https://nodejs.org/.

  • primer paso

    • La Angular CLI (Command Line Interface) es una herramienta esencial para crear y gestionar proyectos Angular. Instálala globalmente ejecutando el siguiente comando en tu terminal:

    npm install -g @angular/cli
  • segundo paso

    • Una vez instalada la CLI, puedes crear un nuevo proyecto Angular con un solo comando. Por ejemplo, para crear un proyecto llamado "mi-primera-app":

    ng new mi-primera-app

    Esto creará una estructura de proyecto completa con todos los archivos necesarios para comenzar.

  • tercer paso

    • Para ver tu aplicación en acción, navega al directorio de tu proyecto y ejecuta el siguiente comando:

    cd mi-primera-app ng serve

    Esto iniciará un servidor de desarrollo local y abrirá tu aplicación en el navegador por defecto (generalmente en http://localhost:4200/).
