    Configuracion


    en este apartado se mostrara el como instalar ionic por cli y algunos comandos para la configuracion de un archivo ionic


    Instalacion

    la ionic cli puede ser instalada con el siguiente comando

    npm install -g @ionic/cli
  • help

    • La CLI de Ionic viene con documentación de comandos que es accesible con la opción --help.


    configuraciones generales

  • files

    • Los valores de configuración se almacenan en archivos JSON. La CLI de Ionic mantiene un archivo de configuración global, usualmente localizado en ~/.ionic/config.json, y archivos de configuración de proyecto, usualmente en el directorio raíz del proyecto como ionic.config.json.
    La CLI proporciona comandos para establecer e imprimir los valores de configuración de los archivos de configuración del proyecto y el archivo de configuración global de la CLI. Ver ionic config --help o ver la documentación para el uso de ionic config get y ionic config set.

  • flags

    • CLI flags son opciones globales que alteran el comportamiento de un comando CLI.

    --help: Instead of running the command, view its help page. --verbose: Show all log messages for debugging purposes. --quiet: Only show WARN and ERROR log messages. --no-interactive: Turn off interactive prompts and fancy outputs. If CI or a non-TTY terminal is detected, the CLI is automatically non-interactive. --confirm: Turn on auto-confirmation of confirmation prompts. Careful: the CLI prompts before doing something potentially harmful. Auto-confirming may have unintended results.
